2. Sifat Limit Fungsi

Sifat-sifat dari limit fungsi yaitu sebagai berikut.

Contoh:

Jika f(x) = 2 maka nilai f(x) saat x mendekati 1 adalah....

Penyelesaian:

Tabel 2.1. Nilai f(x) = 2 saat x mendekati 1

x ... 0,89 0,9 0,98 0,999 ... 1 ... 1,001 1,01 1,1 1,2 ...
f(x) ... 2 2 2 2 ... ... 2 2 2 2 ...

    Dari tabel diatas saat x mendekati 1 dari kiri dan kanan maka nilai f(x) akan mendekati 2. Sehingga  =  = 2, maka berdasarkan definisi  = 2.


Contoh:

Jika nilai f(x) = x maka nilai f(x) saat x mendekati 1 adalah....

Tabel 2.2. Nilai fungsi f(x) saat x mendekati 1

Dari tabel diatas diperoleh nilai  dan  adalah 1, sehingga nilai  = 1.
